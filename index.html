深空之眼 半自動腳本

基於 FindText 圖像辨識的半自動戰鬥腳本。支援多角色技能判定、視窗調整、座標校正，讓你的日常刷關更加輕鬆。

下載最新版本 GitHub 原始碼

核心功能

專為深空之眼設計，涵蓋戰鬥、辨識、視窗與維護的完整工具鏈。

⚔️

半自動戰鬥

自動普攻循環與技能判定，支援 F1 一鍵開關，讓你專注於走位與策略。

🔍

FindText 圖像辨識

採用 FindText 函式庫取代傳統 ImageSearch，在複雜 UI 下提供更穩定的辨識效果。

🖥️

視窗自動調整

F2 一鍵將遊戲視窗調整為 1600×900，並內建進程監控與座標轉換機制。

📐

座標校正工具

內建視覺化座標調整工具，支援拖拽、即時預覽、JSON 配置保存，免去手動改檔。

🔄

自動更新檢查

啟動時自動檢查 GitHub Releases，有新版本立即通知，保持腳本始終最新。

🧩

模組化架構

Config、Window、UI、Update 各模組職責分明，易於維護與擴充新角色。

已支援角色

持續新增中，所有角色均使用 FindText 或 ImageSearch 進行技能判定。

緋染 FindText 巧构 FindText 魂羽 FindText 庚辰 FindText 武羅 FindText 詩蔻蒂 FindText

角色技能素材位於 assets/characters/角色名/，可自行擴充。

📦 使用者版本（推薦）

解壓縮即可使用，內含所有必要檔案，無需安裝開發環境。

  1. 下載最新版本的 releases/ 壓縮檔
  2. 解壓縮到任意資料夾
  3. 執行 run-sid-ag.cmd（或雙擊 sid-ag.ahk）
  4. 按下 F2 調整遊戲視窗
  5. 按下 F1 開始自動攻擊

🛠️ 開發者版本

完整原始碼，適合想修改功能或參與開發的使用者。

  1. git clone 或下載 ZIP
  2. 安裝 AutoHotkey v2
  3. 執行根目錄的 sid-ag.ahk
  4. 或使用批次檔 run-sid-ag.cmd

常見問題

初次使用前建議先閱讀，多數問題都能在這裡找到解答。

請確保所有檔案都在同一目錄，不要單獨移動任何檔案。使用 run-sid-ag.cmd 啟動可自動修正路徑。
請依序檢查：
1. 確認 lib/FindText.ahk 存在
2. 確認角色是否已支援 FindText（見上方角色列表）
3. 確認遊戲視窗標題為 " AetherGazer"
4. 確認桌面解析度 1920×1080、DPI 100%
使用者版本已包含編譯好的 sid-ag.exe，無需安裝 AutoHotkey 即可執行。開發者版本則需要安裝 AutoHotkey v2。
腳本需要遊戲內的「鍵位提示」功能才能正確判斷戰鬥狀態。
設定路徑：遊戲內 → 設定 → 界面 → 開啟鍵位提示。
參考圖片請見 此圖。
使用內建的座標調整工具：執行 CoordinateAdjustmentTool.ahk，按 F8 進入調整模式，在遊戲視窗上拖拽紅色框到正確位置，按 F9 保存即可。
